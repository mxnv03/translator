Проект "My translator"

С помощью данного проекта можно перевести текст на 5 языков: английский, русский, испанский, французский, немецкий.

Как пользоваться:
1. Запустите файл "main.py"
2. В терминале нажмине на ссылку - "Running on..."
3. В полe "Set the language you want to translate:" введите название языка с большой буквы, на который вы хотите перевести ваш текст.
4. В поле "Set the language from which you want to translate:" введите название языка с большой буквы, с которого вы переводите.
5. Введите текст в поле "Enter your text:".


В данном проекте были использованы библиотеки Flask, Translator
